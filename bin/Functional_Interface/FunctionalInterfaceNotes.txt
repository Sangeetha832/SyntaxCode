-functional interfaces - its follow the abstract method inside the functional interface.

	@FunctionalInterface
	interface Operation{
	public double calc(double num1, double num2);
	}

- abstract method - it contains method definition without body of the method.
	public double calc(double num1, double num2);

Functional Interface Type:
	- have only one abstract method, one or more default method.
	- User defined functional interface

		@FunctionalInterface
		interface operation{
		public double calc(double num1, double num2);
		}

	- Inbuild functional interface
		- function - single input, single return value
			Function<Long, Long> addnum = (value)-> value+20;
		- predicate - it represent function takes single input, return true or false.
			predicate<Integer> result = (age)-> age > 18;
		- supplier - it represent function without taking any input
			Supplier<Integer> randomNum = ()->new Integer((int) Math.random*(100));
		- consumer - it represents function process a values without returning anything
			Consumer<String> printVal = (name)->System.out.println(name);
